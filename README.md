# MediMind - 智能医疗助手小程序

一个基于Taro + React + TypeScript开发的智能医疗助手微信小程序，提供AI驱动的健康咨询和症状分析服务。

## 🎯 项目简介

MediMind是一个专为微信小程序平台设计的智能医疗助手，通过AI技术为用户提供便捷的健康咨询和症状分析服务。项目采用现代化的前端技术栈，确保良好的用户体验和代码质量。

## ✨ 当前功能

### 已实现功能
- **AI智能问诊** - 基于大语言模型的对话式健康咨询
- **响应式界面** - 完美适配各种手机屏幕
- **主题切换** - 支持深色/浅色模式
- **底部导航** - 自定义TabBar组件
- **流畅动画** - 优雅的过渡和交互动效

### 页面结构
- **聊天页面** - 主界面，提供AI问诊对话
- **个人中心** - 用户信息和设置管理

## 🛠 技术栈

### 核心框架
- **Taro 4.1.4** - 跨端小程序开发框架
- **React 18** - 组件化UI开发
- **TypeScript** - 类型安全的JavaScript

### 样式方案
- **SCSS** - CSS预处理器
- **CSS3动画** - 流畅的交互动画
- **Flexbox布局** - 响应式设计

### 开发工具
- **ESLint** - 代码质量检查
- **Stylelint** - 样式规范检查
- **Husky** - Git钩子管理
- **Commitlint** - 规范化提交信息

## 🚀 快速开始

### 环境准备
- Node.js >= 16.0.0
- 微信开发者工具（最新版）
- Git

### 安装依赖
```bash
npm install
```

### 开发运行
```bash
# 微信小程序开发模式
npm run dev:weapp

# 构建微信小程序
npm run build:weapp
```

### 其他平台
```bash
# H5开发模式
npm run dev:h5

# 支付宝小程序
npm run dev:alipay

# 百度小程序
npm run dev:swan
```
